  • Location:

3. Check of Equipment

The following items shall be checked and recorded in the commissioning report for all hardware supplied by Miros. For each installed unit of equipment record the following:

  • Location:
    1. Check that equipment has adequate air flow to avoid overheating (especially when installed in closed cabinets etc.).
    2. Verify that distance to magnetic compass is in accordance with compass safe distances.
    3. Verify that radar scanner unit position is satisfactory for system performance. Check obstacle free view of ocean surface in sector for Wavex default measure area. Record any obstacles within sector in the Location section for Radar Specific checks.
  • Visual:
    1. Check for physical damage of the equipment (i.e., caused by accidents, improper or rough handling of equipment during installation or transport). Corrosion, paint damage etc...
    2. Verify that equipment is labelled with product no, serial no, tag no etc. as relevant.
  • Mechanical:
    1. Check that equipment is properly mounted and otherwise correctly installed.
  • Cabling:
    1. Check that signal cables are correctly installed according to wiring diagrams and have been properly secured in place.
  • Power:
    1. Check that incoming power supply voltage and polarity is correct.
  • Earthing:
    1. Verify that equipment utilizing a signal earth is properly connected to a proper earth terminal

5. Startup

Computer & Software Installation Checks

The following items shall be checked and recorded in the commissioning report for all hardware supplied by Miros. For each installed unit of network connected equipment record the following:

  • Computer startup:
    1. Verify that the computer has booted correctly.
  • System requirements:
    1. Verify that the system computer meets latest Wavex SW requirements as stated in “PR-017 - Miros X-band suite computers (Requirement specification)” [7].
  • Windows version:
    1. Open command prompt and run the 'ver' command.
  • Wavex software version:
    1. Look up Wavex version in: Control Panel\Programs\Programs and Features. If newer Wavex SW version is available, run latest WavexInstaller.msi to upgrade to latest version.
  • Wavex software startup:

      PARAGRAPH HERE Ensure the following SW applications have started after the computer power-on:

    1. Wavex configuration manager
    2. MirAdm04
    3. MirRap01
    4. MirSip12
    5. MirSip30
    6. MirAdm14
    7. MirPro10

      8. Ensure the following services have started after the computer power-on through the Services desktop app:

    8. Miros Cloud Service
    9. Miros G4 Monitoring
    10. Miros NMEA input
    11. Miros NMEA output
    12. Miros System Monitor
    13. Miros Wavex Web Server
  • Wavex software options:
    1. Verify that license is valid by checking the Wavex status page; localhost/status-page. Note that this will require a working internet connection unless your device will be an offline device. Refer to Appendix B on page 24 if you need to activate the license.
    2. Verify installed SW match license. Record included options.
    3. Access to Vessel and Weather page comes with full license. Access to Wave System option requires license with all options included as well as an on-site available RangeFinder sensor.
  • Third party software start-up:
    1. Verify that any third party applications or tools are running as required.
  • Storage space:
    1. Check that there is enough space on hard drives to meet requirements stated in PR017 - Miros X-band suite computers (Requirement specification)” [6].
  • Serial mouse script:
    1. Run the DisableSerialMice script. This ensures automatic detection of a serial mouse is disabled for all serial ports.
  • Remote access tool:
    1. If remote access will be available, verify that remote access can be made to the computer.
  • Network interface configuration:

      Using Network and sharing center, check properties of network interfaces used to receive or transmit data, do the following:

    1. Disable "Client for Microsoft Networks".
    2. Disable "File and Print Sharing for Microsoft Networks".
  • Windows firewall configuration:

      Ensure that the firewall is configured to let all software modules and services receiving or exporting data via network connections.

    1. Verify that Wavex software applications have the correct rules set here: Control Panel\System and Security\Windows Defender Firewall\Allowed apps.
    2. Go to advanced settings under Control Panel\System and Security\Windows Defender Firewall. Check that inbound and outbound rules for Wavex software applications allows the expected network traffic.
    3. Virus protection:
      1. Ensure no Wavex application or service has been blacklisted by the virus protection program. The settings for Windows Virus & threat protection are found by searching for “Virus” on the Windows search bar
